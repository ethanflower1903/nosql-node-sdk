Oracle NoSQL Database Node.js SDK

This is version 5.3 of Oracle NoSQL Database Node.js SDK for The Oracle NoSQL Database. It supports both the Oracle NoSQL Database Cloud Service and the on-premise product, Oracle NoSQL Database.

To get started, please see the following:

  • Connecting an Application to Oracle NoSQL Database Cloud Service guide describes how to install and configure the SDK to use with Oracle NoSQL Database Cloud Service as well as how to connect your application to the service.
  • Connecting an Application to On-Premise Oracle NoSQL Database guide describes how to install and configure the SDK to use with On-Premise Oracle NoSQL Database as well as how to connect your application to the database.
  • Working With Tables guide describes how to use the SDK to work with NoSQL Database tables in your application.

The rest of the documentation is the API reference for Oracle NoSQL Database Node.js SDK.
